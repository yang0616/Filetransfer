# STORAGE
  在Linux系统上使用C语言的个人项目；此项目是基于tcp传输实现服务器对客户端的命令响应；利用多线程、socket、fock、execv系列函数、文件的md5值、Makefile等实现文件上传、文件下载、VIP闪电上传下载、断点续传等功能。
